# Node boilderplate

A starting place for building applications

## Pre-requisits

You must have node 4.3.0 and NPM 3.3.0 or above.

## Setup

Install node modules

```bash
# install node modules
$ npm install
```

## Tests

```bash
# run tests
$ npm test
```

## Running the application

```bash
# run the application
$ npm run dev
```
